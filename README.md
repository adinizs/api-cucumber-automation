# Automação de APIs

## Dependencies

* ruby 2.5.x ou superior

## Instrucoes para instalacao

#### Instalar o bundler
Para começar, instale o bundler:

```
  gem install bundler
```

```
  bundle install
```


### Executar o teste

O projeto usa o Cucumber para executar seus testes, você pode usar os seguintes comandos para executar:

```
  cucumber -t @tag
```
